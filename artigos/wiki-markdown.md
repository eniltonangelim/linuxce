# Markdown

Markdown é uma ferramenta que converte texto para html, permitindo que você crie conteúdo estilizado usando apenas o formato de texto plano.

Duas coisas sobre o termo _Markdown_ precisam ser esclarecidas:

+ Markdown é um padrão de formatação;
+ Markdown é uma ferramenta que converte texto plano para HTML;

## Objetivo

O objetivo primordial de design para a sintaxe de formatação do _Markdown_ é torná-la o mais legível possível. A idéia é que um documento formatado com Markdown deve ser publicável como-é, como texto sem formatação, sem parecer que ele foi marcado com tags ou instruções de formatação.

## Inspiração

A inspiração para a sintaxe do Markdown é o formato do e-mail.
