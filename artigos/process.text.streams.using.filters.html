Processar stream de texto usando filtros

O candidato deve ser capaz de aplicar filtros

Objetivo

Enviar arquivos de texto e stream de saída através de filtros para modificar o conteúdo usando comandos UNIX do pacote textutils

Termos e Utilitários

  • cat
  • cut
  • expand
  • fmt
  • head
  • join
  • less
  • nl
  • od
  • paste
  • pr
  • sed
  • sort
  • split
  • tail
  • tr
  • unexpand
  • uniq
  • wc

Concatenando arquivos e imprimindo-os na saída padrão

O comando cat tem como principal funcionalidade a impressão do conteúdo contido no arquivo, a partir dessa funcionalidade você consegue realizar os procedimentos a seguir:

  • Exibe o texto na tela
  • Ler arquivos
  • Cria um novo arquivo
  • Concatenar os arquivos
  • Modificar um arquivo

Caso 1: Exibe o texto na tela

Para o nosso primeiro caso, vou exibir o conteúdo do arquivo /proc/$$/limits

cat /proc/$$/limits
